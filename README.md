# 🌙 Moon Mission Data Analysis (Power BI Project)
I created a custom dataset on moon missions, analyzing how many attempts were made by different countries to reach the moon, along with the overall outcomes of those attempts, using Power 


## 📌 Project Overview
This project analyzes the history of **Moon missions** using interactive dashboards in **Power BI**.  
It highlights information about different **countries, space agencies, mission attempts, and success rates** over time.

## 📊 Key Insights
### 🌍 Country Participation
- The **United States** and **Soviet Union/Russia** have the highest number of missions.  
- Countries like **China** and **India** are becoming more active in recent years.  

### 🚀 Mission Success & Failures
- Many missions in the early years (1950s–1970s) failed due to limited technology.  
- After 2000, success rates improved as technology advanced.  

### 🛰️ Space Agency Contributions
- **NASA** leads in Moon exploration.  
- **Roscosmos**, **CNSA (China)**, and **ISRO (India)** are important contributors.  

### 📅 Timeline Trends
- Visual representation of missions per decade.
-Spike during the 1960s–1970s “Space Race”, followed by renewed activity after 2000 with modern players.
